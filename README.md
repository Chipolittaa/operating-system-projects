# Operating Systems Projects
Linux, operating systems and low-level programming projects focused on system internals, security mechanisms and performance analysis.

---

## About
This repository contains practical projects, experiments and research related to Linux systems programming, operating systems and security mechanisms.

Main areas covered:
- Linux system programming;
- Operating systems internals;
- Process and memory management;
- Linux security modules;
- Network packet filtering;
- Performance and scheduler analysis;
- File system benchmarking;
- Low-level Linux development;
